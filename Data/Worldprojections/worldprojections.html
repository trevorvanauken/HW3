In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

To display the map in different projections, I clicked on the bottom right where it allows you to select different CRSs. I then selected the various different projections needed plus 2 others. To see how the projections changed area, shape, etc, I used the indacatrix mapper to add circles/ ovals to see distortion. To export my projections, I used the QGIS layout manager to convert them into PNGs and to add labels.
